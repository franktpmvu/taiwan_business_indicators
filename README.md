# Taiwan Business Cycle Strategy Backtest (台灣景氣燈號投資策略回測)

這是一個基於國發會景氣對策信號與 0050 (元大台灣50) 的投資策略回測工具。本專案透過 Python 自動抓取政府公開資料，並實作「動態資產配置」模型，測試在不同景氣燈號下的投資報酬率。

## 🚀 核心功能
- **自動資料抓取**：自動從國發會 API/Excel 下載最新的景氣燈號數據。
- **策略參數化**：支援自定義股票/現金配置比 (如 10/90 策略)。
- **動態進出場**：支援「藍燈加碼、紅燈避險」等戰術操作。
- **專業報表**：自動產出年化報酬 (CAGR)、最大回撤 (MDD) 報告。

## 📈 策略範例：80/20 模型
- **平時**：每月投入預算的 80% 於 0050，其餘 20% 轉入現金池。
- **藍燈 (景氣低迷)**：觸發訊號，將累積的現金池 100% 買入 0050。
- **紅燈 (景氣過熱)**：根據設定執行持股續抱或分批停利。

## 🛠️ 安裝與使用
1. 安裝環境：`pip install requests pandas yfinance numpy cloudscraper xlrd lxml html5lib playwright openpyxl html5lib matplotlib`
2. 執行主程式：`python stock.ipynb`
3. 應注意的是國發會下載的xls檔案有時候無法被python系統讀取，此時可以手動打開該檔案後另存新檔就能正常讀取。

## 📊 免責聲明
本專案僅供研究與學習使用，不構成任何投資建議。投資一定有風險，過去績效不代表未來表現。
